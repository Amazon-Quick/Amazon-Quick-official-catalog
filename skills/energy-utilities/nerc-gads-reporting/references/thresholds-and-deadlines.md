# Reporting Thresholds, Deadlines, and Authoritative Sources

## Rule Zero applies to this file

Every value in the tables below is time-sensitive and set by NERC. Reporting
thresholds, effective dates, and submission deadlines change as NERC revises its
program. Do not treat the values here as current. Verify each one you rely on
against the authoritative source listed below during the session (web_search or
url_fetch) before using it in classification, validation, or messaging. If a
source is unreachable and the user has not supplied the value, say so and ask.

## Mandatory Reporting Thresholds (verify before use)

| Unit Type | Threshold (verify) | Effective Date (verify) |
|-----------|--------------------|-------------------------|
| Conventional | >= 20 MW | Jan 1, 2013 |
| Wind | phased by aggregate plant size (200 MW+, then 100 MW+, then 75 MW+) | 2018 onward, phased |
| Solar | >= 20 MW aggregate (per NERC GADS-SOL) | Jan 1, 2024 |

The source skill contained conflicting wind and solar effective dates. Resolve
the current thresholds and dates from the NERC GADS DRI and program pages before
telling a user which units must report.

## Submission Deadlines (verify before use)

| Submission | Deadline (verify) |
|-----------|-------------------|
| Q1 (Jan-Mar) | Mid/late in the same year |
| Q2 (Apr-Jun) | Mid/late in the same year |
| Q3 (Jul-Sep) | Mid/late in the same year |
| Q4 (Oct-Dec) | Early in the following year |
| Design Data Updates | Within 30 days of change (verify) |

Confirm the exact calendar deadlines for the applicable reporting year from the
NERC GADS DRI before stating them to the user.

## Authoritative Sources

- NERC GADS Data Reporting Instructions (DRI): https://www.nerc.com/pa/RAPA/gads/Pages/DataReportingInstructions.aspx
- NERC GADS statistical brochures (annual benchmarking data, including industry-average and top-quartile EFOR): https://www.nerc.com/pa/RAPA/gads/Pages/Reports.aspx
- IEEE Standard 762 - Definitions for Use in Reporting Electric Generating Unit Reliability, Availability, and Productivity
- eGADS User Guide (hosted by PJM): https://pjm.com/markets-and-operations/etools/egads
- NERC Section 1600 Data Request (mandatory reporting authority)
- NERC GADS Wind DRI and Solar DRI (supplemental instructions for renewable units)
